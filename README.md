# cli-template-nodejs

Production-grade CLI boilerplate using oclif and TypeScript.

## Quick Start

```bash
npm install
npm run build
npm test
```

## Commands

- `hello` - Example hello command

## Development

```bash
npm run dev       # Run in development
npm run build     # Build TypeScript
npm test          # Run tests
npm run lint      # Lint code
npm run format    # Format code
```

## Publishing

```bash
npm publish
```
